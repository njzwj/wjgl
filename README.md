反复重构后第五次写软件render。仅供学习分享。

需要make、gnu gcc。

顶点着色演示程序：
```bash
make test                   # output: test.png
```

纹理演示程序：
```bash
make demo_texture           # output: demo_texture.png
```

如何使用？请移步`demo`文件夹下的程序。

# 技术特性

1. 只使用了标准库。不依赖第三方库。

1. 目前为止支持顶点着色模式、纹理。

1. 支持齐次裁剪空间中z=0平面的裁剪。可扩展成6面裁剪（利用现有代码修改后即可）。
